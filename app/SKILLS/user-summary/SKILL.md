# User summary Skill

This skill ensures that all values conform to the schema of `user_1_summary.csv`.


## Instructions

All values must conform to this list:

1. user_id
2. weight_kg: weight in kilograms
3. height_cm: height in centimeters
4. age_group: min/max age range
5. gender: Male or Female
6. Day of the week: Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday
7. date: date of the day
8. resting_heart_rate: resting heart rate for the day
9. heart_rate_variability: heart rate variability
10. steps: number of steps for the day
11. rem_sleep_minutes: REM sleep in minutes
12. deep_sleep_minutes: deep sleep in minutes
13. awake_minutes: awake time in minutes
14. light_sleep_minutes: light sleep in minutes
15. bed_time: time going to bed
16. wake_up_time: wake up time
